The Mannual For GuessWordGame: Mission Moon

Instructions:


This game is developed for entertainment and educational purpose.
Please read it carefully.

1. Click on play.
Some random word will be generated for player with category
(e.g. animal, fruit, vegetables etc. word) clue on it.
(e.g. "The word is an animal" )

2. Guess the word.
Now, Guess the word letters according to blanks given for word.

3. If Guess Matches.
The right letter will be display on blanks. If every guess matches,
player will win the game. Player will be on moon.

4. If Guess Doesn't Matches.
Player will have 5 limited lives for guessing right word. If guess
doesn't matches then player will lose 1 life each for every each
wrong guesses and when player runs out of life. Game Over.
Mission Unsuccessful.
